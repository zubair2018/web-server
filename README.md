# Node.js Web Server Assignment

## Description
This project is a basic Node.js web server built using the HTTP module.

## Features
- Routing (/home, /about, /contact)
- Custom 404 page
- CSS styling
- Proper status codes
- Modular file handling

## How to Run
1. Install Node.js
2. Run: node server.js
3. Open browser: http://localhost:3000/home
